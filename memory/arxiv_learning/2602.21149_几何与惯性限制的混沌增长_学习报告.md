## 核心观点

### 主要贡献
论文建立了**经典多体系统中最大Lyapunov指数的显式、Hamiltonian特定的上限**，这些约束来自Hamiltonian流的瞬时稳定性约束，用惯性尺度和相互作用的曲率表达。

### 两类约束

1. **不可违反边界（Non-violable bounds）**
   - 由最坏情况的局部曲率尺度和惯性控制
   - 对空间结构不敏感
   - 形式：λmax ≤ 1/(2m) + ½‖∇²V‖

2. **遍历天花板（Ergodic ceilings）**
   - 保留光谱信息
   - 编码集体模式和有限尺寸效应
   - 在一维耦合转子链中：
     - λmax ≲ (1/2m)[1 - (4π²J/N²)·(I₁(βJ)/I₀(βJ)]

## 从混沌理论角度解读

### 1. Lyapunov指数量化混沌
- Lyapunov时间 τL = 1/λmax 衡量初始条件记忆的丧失
- 惯性项 1/(2m) 代表"惯性瓶颈"：有限惯性限制相空间轨迹对力的响应速度

### 2. 几何与惯性的相互作用
- **局部曲率** ∇²V 控制瞬时拉伸率
- **集体模式**（long-wavelength modes）抑制混沌增长
- 有限尺寸效应在低维系统中显著（1/N² 缩放）

### 3. 热力学极限下的行为
- N → ∞ 时，遍历天花板趋近于惯性天花板 τL = 2m
- 变得独立于温度和相互作用强度
- 这暗示了"微观时间尺度"的存在，限制长期记忆丢失

### 4. 动态不可达区域
- 在 Lyapunov指数-能量平面中存在动态不可达区域
- 遍历天花板比不可违反边界更紧致
- 数值验证确认了理论预测
